REST API - 

GET /users - get all users in HTML - Done
GET /api/users - get all users in JSON - Done
GET /api/users/1 - get users with id 1 - Done

POST /api/users - Create new user

PATCH /api/users/1 - edit user with id 1

DELETE /api/users/1 - delete the user with id 1

+tip - generate mock data with mockaroo.com
